SKILLSYNC AI — DEMO SCRIPT
A 3-4 Minute Spoken Narration
============================================================

[0:00 — HOOK]

Every developer has been there. You've been working hard, building projects, solving
LeetCode problems, pushing code to GitHub. But when it comes time to apply for jobs,
you don't know where you actually stand. What skills are you missing? What should you
learn next? What would make your resume actually pass an ATS filter?

That's the problem SkillSync AI solves.

[0:20 — WHAT IT IS]

SkillSync AI is an AI-powered career intelligence platform built on Google's Agent
Development Kit — ADK 2.0. It belongs to the Agents for Business track. You give it
your resume, your GitHub username, and your coding profile, and it gives you back a
full career intelligence report plus a personalized learning roadmap.

(pause)

It's not a single AI. It's a coordinated team of specialized agents working together.

[0:40 — SHOW THE BANNER]

(show assets/cover_page_banner.png)

SkillSync AI. Automated. Secure. Intelligent.

At its core, this is what career intelligence looks like when you hand it to a
well-designed multi-agent system.

[1:00 — ARCHITECTURE]

(show assets/architecture_diagram.png)

Let me walk you through what happens when you submit your resume.

First, your input goes through the Security Checkpoint. This node scrubs personal
information like email addresses and phone numbers, detects prompt injection attempts,
and validates that the input is actually a resume. If anything is suspicious, execution
stops right there.

If the input is safe, it moves to the Orchestrator. The Orchestrator is a coordinator
agent — it doesn't do the analysis itself. Instead, it delegates to two specialized
sub-agents using AgentTool.

The Resume Analyzer looks at your resume content and computes an ATS compatibility
score from 0 to 100, identifies formatting and wording issues, and lists missing
skills.

The Profile Analyzer uses the MCP Server — that panel on the right — to fetch your
GitHub statistics and coding platform performance. It calls fetch_github_profile and
fetch_coding_profile tools to get your actual data.

The Orchestrator takes those two reports and synthesizes them into a single unified
analysis.

Then, execution pauses. This is the Human-in-the-Loop node. You see the draft report
and you decide: approve it, or give feedback to refine it. The agent waits for you.

Once approved, the Roadmap Generator takes over. It uses the search_learning_resources
MCP tool to find real courses, books, and tutorials for each skill gap, then produces
a step-by-step learning plan with project ideas and interview practice questions.

(pause)

The final output is a beautifully formatted report you can act on immediately.

[2:00 — LIVE DEMO]

(switch to playground at http://localhost:18081)

Let me show you Case 1 — the full happy path.

I'm going to send this input — a software engineer with Python and SQL experience,
a GitHub profile, and a LeetCode account.

(paste the input and submit)

You can see the Security Checkpoint cleared the input. It redacted the email address
I included, and logged an INFO audit event. Now the Orchestrator is delegating to
the Resume Analyzer and the Profile Analyzer in parallel.

(wait for HITL pause)

Here's the Human-in-the-Loop pause. The system is showing me a draft report with an
ATS score. I'll type "yes" to approve it.

(type "yes")

Now the Roadmap Generator is running, using the MCP server to find resources for
each skill gap. And here's the final output — a complete career intelligence report.

(pause, let the audience see the output)

Now let me quickly show you Case 2 — the security path.

(submit injection payload)

Immediately blocked. No LLM calls were made. The security checkpoint detected the
phrase "ignore previous instructions" and returned a security violation.

[3:00 — SECURITY AND MCP]

The Security Checkpoint uses a three-layer defense: prompt injection keyword
detection, PII regex scrubbing, and domain validation. Every decision generates
a structured JSON audit log to stderr — fully auditable.

The MCP Server exposes three tools via FastMCP and stdio transport. It runs as an
independent subprocess, which means tool failures can't crash the main agent. The
same server can be reused by other agents in the future.

[3:30 — IMPACT AND CLOSE]

SkillSync AI matters because personalized career guidance has always been expensive
and inaccessible. A good career coach can cost hundreds of dollars per session.
SkillSync AI makes that intelligence available on demand, to anyone, for free.

The security-first design means it can be deployed in professional environments where
candidate data must be handled carefully.

And the human-in-the-loop design means it's collaborative, not a black box. You stay
in control.

This is what Google ADK 2.0 makes possible — a team of intelligent agents, working
together, with guardrails built in from the start.

Thank you.
